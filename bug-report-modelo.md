# Modelo de Bug Report

## Título
Erro ao cadastrar leitura de consumo

## Descrição
Sistema retorna erro 500 ao enviar dados válidos.

## Passos para reproduzir
1. Acessar endpoint POST /leituras
2. Enviar JSON válido
3. Executar requisição

## Resultado esperado
Registro criado com status 201

## Resultado obtido
Erro 500

## Ambiente
Windows 11 / Postman / API local

## Severidade
Alta
