# QA Portfolio – Michele Telles

Repositório com exemplos práticos de atividades de Qualidade de Software (QA), incluindo documentação de testes, registro de bugs e checklists.

Este material demonstra minha organização, atenção aos detalhes e aplicação de boas práticas de teste em sistemas web e APIs.

## Conteúdo
- Casos de teste manuais
- Modelo de relatório de bugs
- Checklist de testes de API
- Checklist de testes web

## Ferramentas utilizadas
- Postman
- Excel
- Git/GitHub
- Testes manuais exploratórios

## Objetivo
Evidenciar minhas habilidades práticas em QA além do código, simulando o dia a dia real de testes em projetos de software.
